Usual loop

Versions use CalVer. The current rewrite version is 26.5.12.

make test

make run-tui

Local launcher

Run make navd-update, then start navd. The launcher builds local binaries and installs only navd into ~/.local/bin.

Set NAV_BACKEND=/path/to/nav-backend when you want the TUI to use a specific backend binary.
