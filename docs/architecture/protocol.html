Protocol

Commands: JSON-RPC 2.0 at POST /rpc (initialize, session.create, session.sendMessage, tool approve/reject, etc.). Responses acknowledge state; long output streams separately.

Events: SSE at GET /sessions/{session_id}/events (message.delta, tool lifecycle, run completed/failed, …). Reconnect with Last-Event-ID.

Transport: local HTTP on 127.0.0.1, random port, local auth token. All protocol IDs are lowercase UUIDv7.
