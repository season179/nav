Full listing

Rust crates live in crates/:

  • nav-backend: binary entrypoint
  • nav-server: HTTP, JSON-RPC, SSE, routing, bootstrap
  • nav-harness: agent engine and model/provider code
  • nav-protocol: shared RPC and event types
  • nav-types: primitives, including UUIDv7 IDs

Go packages live in tui/:

  • cmd/nav: released TUI command
  • cmd/navd: local dev entrypoint
  • internal/app: composition root
  • internal/client: protocol boundary
  • internal/ui: Bubble Tea views and styles
  • internal/localdev: build/run support for navd
