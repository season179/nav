Crates and folders

Dependency flow: nav-backend → nav-server → nav-harness, with nav-protocol → nav-types for wire types.

  • nav-backend — binary entrypoint
  • nav-server — HTTP, JSON-RPC, SSE, auth, bootstrap
  • nav-harness — agent engine (sessions, tools, models, guardrails, …)
  • nav-protocol — shared RPC and event types
  • nav-types — primitives (UUIDv7 IDs)
  • tui/ — Go Bubble Tea frontend (cmd/nav, internal/*)
